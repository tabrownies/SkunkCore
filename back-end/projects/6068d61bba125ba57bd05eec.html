Scream Machine

While still in the works, this machine will be used to prank some of our friends. It will be built using a Raspberry Pi. Here is how it works.

  1. We will prepare it by typing the wifi name and password into a simple text file.
  2. After placed and plugged in, the Raspberry Pi will boot up and begin the scripts.
  3. The first script, programed in Python, will identify the wifi network provided and connect to it.
  4. The second script, a Node.js server, will serve a website onto the internet.
  5. After ensuring everything is working, the server will send an API call to my computer. This API call will provide the website to connect to.
  6. After accessing the website, we will click the Initiate Scream button, and the speaker will scream.

Current Progress:

The screaming script works. It serves a webpage and can be initiated by a seperate device visiting the webpage. It remains to develop the Python script.
